2. Finalidades del Tratamiento

  • Cumplir con obligaciones legales y fiscales

3. Base Legitimadora

El tratamiento de tus datos se basa en:

  • Consentimiento: Para el envío de comunicaciones comerciales
  • Ejecución contractual: Para la prestación de servicios solicitados
  • Interés legítimo: Para responder a consultas y mantener relaciones comerciales
  • Cumplimiento legal: Para obligaciones fiscales y contables

4. Destinatarios de los Datos

Tus datos podrán ser comunicados a:

Puedes ejercer los siguientes derechos:
